Orbital Brand Manifesto

Identidad viva de Julián Zuluaga y Orbital Lab. Una constelación donde la IA, el diseño y la docencia se alinean para que ver sea entender.

ADN de marca Sistema Visual Roadmap

01 · ADN Y ARQUETIPOS

MANTRA

Desarrollar inteligencia que vea, piense y transforme.

Todo proyecto debe superar el filtro: ¿ayuda a que las personas comprendan mejor el mundo que los rodea?

PERSONALIDAD

Visionario · Técnico · Empático · Rebelde Elegante

Creador Sabio Storyteller Ethical AI

La marca es una órbita entre la ciencia aplicada, la estética de datos y la narrativa humana. Docencia, laboratorio y producto no son ramas aisladas: son satélites que comparten un mismo núcleo.

02 · PILARES

TECNOLOGÍA VISIBLE

Interfaces orbitales, HUDs, mallas de datos y circuitos vibrantes muestran el interior de cada decisión algorítmica.

MINIMALISMO DE PODER

Negro profundo + rojo orbital = foco absoluto. El vacío no es ausencia, es territorio para el contenido.

NARRATIVA HUMANA

Deportistas, analistas, estudiantes y ecosistemas. Siempre existe un rostro que siente los efectos de la IA.

APRENDIZAJE APLICADO

Guías interactivas, notebooks y superagentes listos para ejecutar. Enseñar es abrir el laboratorio.

05 · DOCENCIA VIVA

Cada clase nace en este repositorio: materias/<materia>/clases/<fecha>-clase-XX-tema/. El site debe ingerir esa estructura y transformarla en experiencias navegables: timeline, tarjetas, buscador.

FORMATOS

  • Guías HTML interactivas (Plotly, diseño orbital).
  • Notebooks ejecutables + datasets documentados.
  • Resumenes pedagógicos y checklists.

VINCULACIÓN

  • Sincronizar metadatos con Content Collections de Astro.
  • Indexar recursos para Fuse.js (búsqueda client-side).
  • Generar badges por fase de CRISP-DM, skill y dificultad.

06 · ROADMAP

  1. Q4 2025: tokens de diseño (`theme.json`), componentes base, prototipo Figma.
  2. Q1 2026: lanzamiento del site (Home + Lab + Docencia), migración de contenidos previos.
  3. Q2 2026: buscador Fuse.js, biblioteca interactiva y agentes que sugieren clases.
  4. Q3 2026: versión bilingüe, micrositios narrativos para cada vertical.
  5. Q4 2026: experiencia inmersiva WebGL con storytelling de proyectos.

07 · ÉTICA Y LEGADO

TRANSPARENCIA

Datos, modelos y limitaciones documentados. Protocolos de privacidad y seguridad claros.

INCLUSIÓN

Datasets representativos, ejemplos diversos, lenguaje no sesgado. IA como herramienta para amplificar voces.

La misión es permanente: enseñar, crear y desplegar IA responsable desde Latinoamérica para el mundo.

08 · MOODBOARD

Referencia visual que guía la paleta, la iluminación y el estilo orbital. Rojo energético, negros profundos, texturas de datos y rostros que conectan la tecnología con las personas.
