🛠️ Instalación de OpenAI Codex

1

Instalar Python 3

Python es el lenguaje que usaremos para minería de datos. Es fundamental tenerlo instalado antes de continuar.

🐍 Instalación de Python:
  1. Ve a python.org/downloads
  2. Descarga la versión 3.10 o superior (recomendado: la última versión estable)
  3. Ejecuta el instalador
  4. ¡MUY IMPORTANTE! Marca la casilla "Add Python to PATH" en la primera pantalla
  5. Haz clic en "Install Now"
  6. Espera a que termine la instalación
⚠️ CRÍTICO: Si no marcas "Add Python to PATH", el agente CLI no podrá encontrar Python. Si olvidaste marcarlo, desinstala Python y vuelve a instalarlo correctamente.
✅ Verificar instalación: Abre PowerShell (presiona Win + X y selecciona "Windows PowerShell") y escribe: python --version. Deberías ver algo como "Python 3.10.x" o superior.
2

Instalar Visual Studio Code

VSCode es el editor de código más popular del mundo. Si ya lo tienes instalado, salta este paso.

⬇️ Descarga e instalación:
  1. Ve a code.visualstudio.com
  2. Descarga la versión para Windows
  3. Ejecuta el instalador y sigue las instrucciones
  4. Acepta todas las opciones por defecto
⚠️ Importante: Durante la instalación, marca la casilla "Agregar a PATH" (generalmente está marcada por defecto).
3

Abrir VSCode

Simplemente abre VSCode en cualquier carpeta o crea una carpeta nueva para trabajar.

🎯 Cómo abrir VSCode:
  1. Abre VSCode desde el menú de inicio de Windows
  2. Usa Archivo → Abrir Carpeta
  3. Selecciona cualquier carpeta (ej: Documentos) o crea una nueva
💡 Nota: No te preocupes por crear carpetas manualmente. El agente CLI creará toda la estructura automáticamente cuando uses el prompt mágico.
4

Instalar la Extensión de Codex

Ahora instalaremos el agente de OpenAI directamente en VSCode.

🔌 Instalación de la extensión:
  1. En VSCode, haz clic en el ícono de Extensiones (cuadrados en la barra lateral izquierda) o presiona Ctrl + Shift + X
  2. En la barra de búsqueda, escribe: Codex OpenAI
  3. Busca la extensión oficial "Codex – OpenAI's coding agent" por OpenAI
  4. Haz clic en Install
5

Autenticarse con ChatGPT

Es momento de conectar la extensión con tu cuenta de ChatGPT.

🔐 Proceso de autenticación:
  1. Después de instalar, VSCode te mostrará un botón "Sign in with ChatGPT"
  2. Haz clic en ese botón
  3. Se abrirá tu navegador web
  4. Inicia sesión con tu cuenta de ChatGPT (la que tiene Plus, Pro, etc.)
  5. Autoriza la conexión
  6. Vuelve a VSCode - ¡ya estará conectado!
⚠️ Importante: Asegúrate de estar usando la cuenta que tiene la suscripción activa de ChatGPT Plus/Pro.
6

¡Usar el Prompt Mágico!

¡Ahora viene la magia! Usa el prompt de instalación automática que está más abajo en este tutorial.

🎯 Pasos finales:
  1. Desplázate hacia abajo hasta la sección "Prompt Mágico: Instalación Automática del Entorno"
  2. Copia el prompt completo usando el botón
  3. Busca el ícono de Codex en la barra lateral de VSCode
  4. Haz clic en él para abrir el panel de Codex
  5. Pega el prompt en el chat
  6. ¡Presiona Enter y observa cómo Codex instala TODO automáticamente!
☕ Relájate: Codex ejecutará todos los comandos necesarios, te explicará cada paso, y configurará tu entorno completo de minería de datos. Esto puede tomar 5-10 minutos dependiendo de tu conexión a internet.

🎓 Instalación de GitHub Copilot (Gratis para Estudiantes)

1

Instalar Python 3

Python es el lenguaje que usaremos para minería de datos. Es fundamental tenerlo instalado antes de continuar.

🐍 Instalación de Python:
  1. Ve a python.org/downloads
  2. Descarga la versión 3.10 o superior (recomendado: la última versión estable)
  3. Ejecuta el instalador
  4. ¡MUY IMPORTANTE! Marca la casilla "Add Python to PATH" en la primera pantalla
  5. Haz clic en "Install Now"
  6. Espera a que termine la instalación
⚠️ CRÍTICO: Si no marcas "Add Python to PATH", el agente CLI no podrá encontrar Python. Si olvidaste marcarlo, desinstala Python y vuelve a instalarlo correctamente.
✅ Verificar instalación: Abre PowerShell (presiona Win + X y selecciona "Windows PowerShell") y escribe: python --version. Deberías ver algo como "Python 3.10.x" o superior.
2

Verificar GitHub Student Developer Pack

Para usar Copilot gratis, necesitas tener el GitHub Student Developer Pack activado.

🎒 ¿Ya tienes el Student Pack?
  1. Ve a education.github.com/pack
  2. Inicia sesión con tu cuenta de GitHub
  3. Si ya lo tienes, verás "You have GitHub Education"
  4. Si NO lo tienes, haz clic en "Sign up for Student Developer Pack"
  5. Necesitarás tu correo institucional (@uexternado.edu.co) y una foto de tu carnet
⏱️ Nota: La verificación puede tomar hasta 1-2 días hábiles. Si aún no lo tienes, aplica ahora y continúa este tutorial cuando sea aprobado.
3

Instalar Visual Studio Code

VSCode es el editor de código más popular. Si ya lo tienes, salta este paso.

⬇️ Descarga e instalación:
  1. Ve a code.visualstudio.com
  2. Descarga la versión para Windows
  3. Ejecuta el instalador y sigue las instrucciones
  4. Acepta todas las opciones por defecto
4

Instalar GitHub Copilot

Ahora instalaremos GitHub Copilot directamente en VSCode.

🔌 Instalación de la extensión:
  1. En VSCode, haz clic en el ícono de Extensiones (cuadrados en la barra lateral) o presiona Ctrl + Shift + X
  2. En la barra de búsqueda, escribe: GitHub Copilot
  3. Busca "GitHub Copilot" oficial por GitHub
  4. Haz clic en Install
  5. También instala "GitHub Copilot Chat" (extensión complementaria)
5

Autenticarse con GitHub

Es momento de conectar Copilot con tu cuenta de GitHub.

🔐 Proceso de autenticación:
  1. Después de instalar, VSCode mostrará una notificación para autenticarse
  2. Haz clic en "Sign in to GitHub"
  3. Se abrirá tu navegador web
  4. Inicia sesión con tu cuenta de GitHub (la que tiene el Student Pack)
  5. Autoriza la conexión de GitHub Copilot
  6. Vuelve a VSCode
✅ Verificación: En la barra de estado inferior de VSCode, deberías ver un ícono de Copilot (parece un logo de GitHub). Si está activo, mostrará un color sólido.
6

Activar el Modo Agente

GitHub Copilot tiene un modo especial de "agente" que es más autónomo y poderoso.

⚙️ Configuración del modo agente:
  1. Presiona Ctrl + Shift + P para abrir la paleta de comandos
  2. Escribe: Preferences: Open Settings (JSON)
  3. Se abrirá un archivo JSON
  4. Agrega la siguiente línea dentro de las llaves {}:
"chat.agent.enabled": true
📝 Tip: Si ya hay otras configuraciones, asegúrate de poner una coma al final de la línea anterior antes de agregar esta nueva configuración.
7

¡Usar el Prompt Mágico!

¡Ahora viene la magia! Usa el prompt de instalación automática que está más abajo en este tutorial.

🎯 Pasos finales:
  1. Desplázate hacia abajo hasta la sección "Prompt Mágico: Instalación Automática del Entorno"
  2. Copia el prompt completo usando el botón
  3. Presiona Ctrl + I para abrir GitHub Copilot Chat
  4. En el menú desplegable "Chat mode", selecciona "Agent"
  5. Pega el prompt en el chat
  6. ¡Presiona Enter y observa cómo Copilot instala TODO automáticamente!
☕ Relájate: GitHub Copilot ejecutará todos los comandos necesarios, te explicará cada paso, y configurará tu entorno completo de minería de datos. Esto puede tomar 5-10 minutos dependiendo de tu conexión a internet.

🚀 Prompt Mágico: Instalación Automática del Entorno ⚡

¡Este es el máximo poder! Copia este prompt y pégalo en tu agente CLI. Él hará TODO por ti: crear carpetas, instalar Python, configurar el entorno, instalar librerías, crear notebooks... ¡AUTOMÁTICAMENTE!

Hola Codex. Estamos en Windows y quiero que me guíes tú ejecutando los comandos necesarios (usa PowerShell) y explicándome qué haces. Objetivo: 1. Asegura que trabajo desde la carpeta raíz `c:\Users\<MI_USUARIO>\Documents\code`. Si no existe, créala y cuéntame. 2. Dentro crea un proyecto llamado `ds-plantilla`. 3. Dentro de `ds-plantilla`, crea un entorno virtual con Python 3 (`py -3 -m venv .venv` o `python -m venv .venv`) y confirma versión. 4. Activa el venv y actualiza `pip`, `setuptools`, `wheel`. 5. Instala estas librerías con pip: `numpy pandas scikit-learn jupyter matplotlib seaborn ipykernel`. 6. Congela dependencias en `requirements.txt`. 7. Crea un `README.md` con instrucciones sencillas para volver a levantar este entorno. 8. Crea la carpeta `notebooks/` y el notebook `01_intro.ipynb` con: - Celda markdown "Bienvenido…". - Celda de código que muestre versiones de Python, OS, NumPy, Pandas. - Celda de código con DataFrame de ejemplo y `df.head()`. - Celda de código con gráfico scatter usando seaborn y matplotlib. 9. Registra el kernel de Jupyter con nombre `ds-plantilla` y display name `Python (ds-plantilla)`. 10. Al final recuérdame cómo activar el venv, abrir Jupyter Lab y seleccionar el kernel en VS Code. Por cada paso: ejecuta el comando, explícame qué estás haciendo, reporta cualquier error y cómo solucionarlo. Si algo tarda, avísame que sigues trabajando. Hazlo todo tú.
✨ ¿Qué hace este prompt mágico?
  • ✅ Crea toda la estructura de carpetas automáticamente
  • ✅ Configura un entorno virtual de Python aislado
  • ✅ Instala TODAS las librerías necesarias para Minería de Datos (NumPy, Pandas, Scikit-learn, Jupyter, etc.)
  • ✅ Crea un notebook Jupyter de ejemplo con código funcional
  • ✅ Configura el kernel de Jupyter para VSCode
  • ✅ Te explica cada paso mientras lo ejecuta
💡 Cómo usar este prompt:
  1. Copia el prompt usando el botón de arriba
  2. Abre VSCode en cualquier carpeta
  3. Abre tu agente (Codex o Copilot en modo Agent)
  4. Pega el prompt completo en el chat del agente
  5. Presiona Enter y... ¡relájate! ☕
  6. El agente ejecutará cada comando, te explicará qué hace, y resolverá errores automáticamente
⚠️ Requisito previo: Asegúrate de tener Python 3 instalado en tu sistema. Si no lo tienes, descárgalo de python.org/downloads (versión 3.10 o superior recomendada). Durante la instalación, marca la casilla "Add Python to PATH".
🎯 Para GitHub Copilot: Si usas Copilot, asegúrate de que esté en modo Agent. Presiona Ctrl + I para abrir el chat, y en el menú desplegable selecciona "Agent" antes de pegar el prompt.
