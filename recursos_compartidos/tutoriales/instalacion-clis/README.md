# Tutorial de Instalación de Agentes CLI

Tutorial interactivo para estudiantes de Minería de Datos de la Universidad Externado de Colombia.

## ¿Qué es esto?

Este es un tutorial HTML completamente interactivo que guía a los estudiantes paso a paso en la instalación y configuración de agentes de IA (Codex o GitHub Copilot) para automatizar la configuración de su entorno de trabajo de ciencia de datos.

## Características

- **Interfaz futurista y atractiva** con animaciones y diseño responsive
- **Dos rutas de instalación**:
  - OpenAI Codex (para quienes tienen ChatGPT Plus/Pro)
  - GitHub Copilot (gratuito para estudiantes con GitHub Student Developer Pack)
- **Sistema de progreso interactivo** con checkboxes que guardan el estado
- **Prompt mágico** que automatiza completamente la instalación del entorno de minería de datos
- **Guardado automático** del progreso en localStorage

## Cómo usar

1. Abre el archivo `tutorial-instalacion-agentes.html` en cualquier navegador moderno
2. Selecciona tu ruta preferida (Codex o GitHub Copilot)
3. Sigue los pasos interactivos
4. Usa el "Prompt Mágico" al final para que el agente instale todo automáticamente

## El Prompt Mágico

El prompt incluido en el tutorial permite que Codex o Copilot:

- Cree automáticamente la estructura de carpetas del proyecto
- Configure un entorno virtual de Python
- Instale todas las librerías necesarias (NumPy, Pandas, Scikit-learn, Jupyter, etc.)
- Cree un notebook Jupyter de ejemplo con código funcional
- Configure el kernel de Jupyter para VSCode
- Genere un README con instrucciones

Todo esto **sin que el estudiante tenga que escribir una sola línea de código**.

## Requisitos previos

- Python 3.10 o superior instalado (con "Add to PATH" marcado)
- VSCode instalado
- Para Codex: Suscripción a ChatGPT Plus/Pro
- Para Copilot: GitHub Student Developer Pack activo

## Estructura del tutorial

### Para Codex (6 pasos):
1. Instalar Python 3
2. Instalar Visual Studio Code
3. Abrir VSCode
4. Instalar la extensión de Codex
5. Autenticarse con ChatGPT
6. Usar el Prompt Mágico

### Para GitHub Copilot (7 pasos):
1. Instalar Python 3
2. Verificar GitHub Student Developer Pack
3. Instalar Visual Studio Code
4. Instalar GitHub Copilot
5. Autenticarse con GitHub
6. Activar el Modo Agente
7. Usar el Prompt Mágico
